# ELEMENTRIS — 游戏框架设计总索引

> [!abstract] 项目概览
> **ELEMENTRIS** 是一款融合「俄罗斯方块」与「塔防」双核玩法的移动端竖屏益智游戏。
> 「5选1方块决策系统（软约束）」驱动俄罗斯方块操作；「消行→子弹飞弹攻击（贝塞尔曲线）」实现方块与塔防的深度联动；「技能格系统（13种塔防技能）」为消行行为叠加战略层次；「底部增行」构成持续棋盘压力。

## 🎯 MVP 验证目标

- [ ] 5选1方块机制（软约束）的趣味性与策略深度验证
- [ ] 消行→子弹→击杀的核心战斗爽感验证
- [ ] 智能目标选取与暴击倍率的实际体感验证
- [ ] 底部增行 + 敌人双线压力的心流平衡验证
- [ ] 技能格（13种）的策略张力与使用场景差异验证
- [ ] 整体「俄罗斯方块 × 塔防」融合的自然感验证
