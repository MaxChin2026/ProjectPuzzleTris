# CONFIG — 干扰方块配置表

> [!info] 说明
> 本表定义干扰方块系统的所有参数，包括形状、投放节奏、预警时间及难度分级。
> 系统设计原则见 [[03-干扰方块与预警系统]]。

---

## 干扰方块形状定义

| 类型 ID | 名称 | 形状描述 | 占格数 | 解锁阶段 |
| ------- | ---- | -------- | ------ | -------- |
| `INT_SINGLE` | 单格干扰 | ■ （1×1） | 1格 | 阶段1 |
| `INT_H2` | 横条干扰 | ■■ （1×2横向） | 2格 | 阶段2 |
| `INT_L` | L形干扰 | ■■/■ （L型3格） | 3格 | 阶段4 |

> [!note] 形状设计约束
> - 干扰方块均为小型形状，**不使用标准7种Tetris形状**
> - 颜色使用深红色调（`#8B1A1A`），与玩家方块高饱和亮色形成强对比

---

## 难度阶段参数表

| 阶段 | 触发条件（存活时间）| 投放间隔 | 预警时间 | 每次投放列数 | 可用形状 |
| ---- | ------------------- | -------- | -------- | ------------ | -------- |
| 阶段1 | 0～120秒 | 40秒 | 3.0秒 | 1列 | `INT_SINGLE` |
| 阶段2 | 120～300秒 | 30秒 | 2.5秒 | 1～2列 | `INT_SINGLE`、`INT_H2` |
| 阶段3 | 300～540秒 | 22秒 | 2.0秒 | 2列 | `INT_SINGLE`、`INT_H2` |
| 阶段4 | 540～780秒 | 16秒 | 1.5秒 | 2～3列 | 全类型 |
| 阶段5 | 780秒起 | 12秒 | 1.0秒 | 3列 | 全类型 |

> [!warning] 数值调整说明
> 以上数值均为初始设计稿，需通过 v0.3 版本实际测试调平。核心体验目标：
> - 玩家在预警期内能够**感到紧张但有时间做出反应**
> - 干扰方块落地带来的棋盘变化应**可被后续操作修复**，而非立即导致绝境

---

## 列选择随机规则

| 参数 | 说明 |
| ---- | ---- |
| 列范围 | 1～10列（全棋盘宽度） |
| 随机规则 | 纯随机，不受当前棋盘状态影响（简单实现，v1.0后可考虑权重倾向高堆叠列）|
| 多列投放 | 同次投放多列时，选取不相邻的列，避免同一区域被集中轰炸 |

---

## 视觉参数

| 参数 | 数值 |
| ---- | ---- |
| 预警标记颜色 | `rgba(255, 80, 0, 0.5)`（橙色半透明） |
| 即将落下闪烁颜色 | `rgba(200, 0, 0, 0.9)`（深红高亮） |
| 干扰方块填充色 | `#8B1A1A`（深红） |
| 干扰方块边框 | `#CC3333`（略亮深红） |
| 预警闪烁频率 | 最后1秒每200ms切换一次透明度（0.9↔0.3） |
