# CONFIG — 关卡配置表

> [!info] 说明
> 本表定义游戏难度阶段的压力节奏参数，包含底部增行速度与敌人波次的联合配置。
> 压力曲线设计原则见 [[04-压力与奖励曲线]]。

---

## 难度进度机制

> [!abstract] 进度触发方式
> 难度阶段以**玩家存活时间**为基准自动推进，每个阶段持续固定时长后进入下一阶段。
> 底部增行速度与敌人波次强度**同步升级**，确保两条压力线均衡加压。

```
游戏进度：
0s ───────── 120s ───────── 300s ───────── 540s ───────── 780s ──────→ 无限
  [阶段1：入门]  [阶段2：适应]  [阶段3：挑战]   [阶段4：高压]   [阶段5：极限]
```

> [!info] 说明
> 本表定义游戏难度阶段的压力节奏参数，包含底部增行速度与干扰方块投放频率的联合配置。
> 压力曲线设计原则见 [[04-压力与奖励曲线]]。

---

## 难度进度机制

> [!abstract] 进度触发方式
> 难度阶段以**玩家存活时间**为基准自动推进，每个阶段持续固定时长后进入下一阶段。
> 底部增行速度与干扰方块投放频率**同步升级**，确保两条压力线均衡加压。

```
游戏进度：
0s ───────── 120s ───────── 300s ───────── 540s ───────── 780s ──────→ 无限
  [阶段1：入门]  [阶段2：适应]  [阶段3：挑战]   [阶段4：高压]   [阶段5：极限]
```

## 分数系统参数

| 行为 | 分数 | 说明 |
| ---- | ---- | ---- |
| 单行消除 | +100 | 基础消行得分 |
| 双行同时消除 | +300 | 双行连消奖励倍率 |
| 三行同时消除 | +700 | 三行连消奖励倍率 |
| 四行同时消除 | +1500 | 四行连消奖励倍率（最大单次消行）|
| 列消除（每格）| +15 | 列消除奖励格触发，按格计分 |
| 奖励行消除 | +100 | 奖励格触发的行消除，同单行消除分 |
| 阶段奖励 | +500/阶段 | 每进入新阶段的存活奖励分 |

> [!note] 分数设计意图
> - 连消奖励系数高于单行，鼓励玩家规划多行同时消除
> - 阶段奖励鼓励长时间存活，与分数系统形成持续激励

## 满棋盘处理

> [!warning] 满棋盘 = 直接 GAME OVER
> 与旧版「阶梯扣血缓冲」不同，新版设计中**棋盘溢出即直接结束本局**。
> 这使得棋盘管理的失误成本更高，强化了压力感，要求玩家主动维护棋盘可持续性。

| 情况 | 处理 |
| ---- | ---- |
| 新方块无法在顶部生成 | 即时触发 GAME OVER |
| 底部增行导致堆叠超出 | 先推高，若方块超出边界则游戏结束 |
| GAME OVER后 | 显示最终分数、本局时长、达到的最高阶段 |
