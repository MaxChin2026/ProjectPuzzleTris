---

## 技能格生成参数

| 参数 | 数值 | 说明 |
| ---- | ---- | ---- |
| 生成概率 | 30% | 每次生成3个候选方块时，30%概率注入1个技能格 |
| 最大同时数量 | 1 | 3个候选方块中同时最多1个技能格 |
| 技能格位置 | 随机选取方块内任意1格 | 不固定为特定格索引 |
| 触发条件 | 含技能格的行被完整消除 | 非放置时触发，而是消行时触发 |

## 技能权重（生成概率分布）

> [!note] 权重设计原则
> 高价值/强效技能出现概率较低，通用/适配性高的技能频率较高，保证游戏体验的多样性。

| 技能ID | 权重 | 出现频率（相对）|
| ------ | ---- | --------------- |
| `SKL_SNIPE` | 15 | 高 |
| `SKL_SLOW` | 15 | 高 |
| `SKL_FREEZE` | 12 | 中高 |
| `SKL_BURST` | 12 | 中高 |
| `SKL_CHAIN` | 10 | 中 |
| `SKL_STUN` | 10 | 中 |
| `SKL_PULL` | 8 | 中低 |
| `SKL_METEOR` | 8 | 中低 |
| `SKL_ARMOR_BREAK` | 6 | 低 |
| `SKL_FIRE_ZONE` | 6 | 低 |
| `SKL_SHIELD` | 5 | 低 |
| `SKL_AMPLIFY` | 5 | 低 |
| `SKL_NOVA` | 2 | 极低 |
| `SKL_CHAIN_CLEAR` | 4 | 低（高价值，低频出现）|

> [!warning] 数值调整说明
> 以上权重和数值均为初始设计稿，需通过 v0.4 版本实际测试调平。
> 核心体验目标：
> - 技能效果玩家**立即可感知**（视觉 + 结果要直观）
> - 技能选择对当前战局**明显有影响**，不同技能在不同场景有不同价值

# CONFIG — 奖励效果配置表

> [!info] 说明
> 本表定义奖励方块的所有参数，包括奖励类型、触发效果、生成概率及视觉标识。
> 系统设计原则见 [[04-压力与奖励曲线#奖励方块机制]]。

---

## 奖励类型定义

| 类型 ID | 图标 | 名称 | 触发效果说明 |
| ------- | ---- | ---- | ------------ |
| `RWD_ROW` | 🔥 | 行消除 | 立即强制消除玩家指定的一行（默认为棋盘最底层不完整行） |
| `RWD_COL` | ⚡ | 列消除 | 立即清空奖励格所在列的全部方块 |
| `RWD_FILL` | ✨ | 填空缺 | 将棋盘最底层不完整行的所有空格填满，若填满则触发消行 |

---

## 触发效果详细规则

### 行消除（RWD_ROW）

| 参数 | 规则 |
| ---- | ---- |
| 目标行选取 | 棋盘中**最靠近底部的不完整行**（含有空格的行） |
| 效果动画 | 该行高亮闪烁后消除，上方内容正常下落 |
| 边缘情况 | 若棋盘全空，则无效果（不消除） |
| 分数奖励 | 视为正常消行，按消行分数规则计算 |

### 列消除（RWD_COL）

| 参数 | 规则 |
| ---- | ---- |
| 目标列 | 奖励格触发时所在的列（水平位置） |
| 效果 | 该列所有非空格子清除，列内容彻底归零 |
| 效果动画 | 该列自上而下消除特效 |
| 分数奖励 | 消除的格子数 × 单格分值（低于整行消行分值，避免过强）|

### 填空缺（RWD_FILL）

| 参数 | 规则 |
| ---- | ---- |
| 目标行选取 | 棋盘中**最靠近底部的不完整行** |
| 效果 | 该行所有空格被填满（填充颜色为中性灰色，无形状归属） |
| 连锁触发 | 填满后立即检测消行，若满行则触发正常消行流程 |
| 边缘情况 | 若棋盘最底层行已满，则向上寻找下一个不完整行 |
| 分数奖励 | 若触发消行，按正常消行分数计算 |

---

## 奖励生成概率参数

| 参数 | 数值 | 说明 |
| ---- | ---- | ---- |
| 每批方块含奖励概率 | 30% | 每次刷新3个候选方块时，有30%概率其中1个含奖励格 |
| 同批最多奖励格数 | 1 | 3个候选方块中最多1个含奖励，避免奖励泛滥 |
| 行消除概率权重 | 40% | 在奖励类型中，行消除占40%概率 |
| 列消除概率权重 | 30% | 列消除占30%概率 |
| 填空缺概率权重 | 30% | 填空缺占30%概率 |

> [!note] 概率设计意图
> - **行消除**权重最高，因其效果最直接（立即消行），是玩家最直观的正反馈
> - **列消除**和**填空缺**权重相近，两者在特定棋盘状况下有独特价值，保持多样性

---

| 参数 | 行消除(🔥) | 列消除(⚡) | 填空缺(✨) |
| ---- | ---------- | ---------- | ---------- |
| 图标大小 | 格子面积的40% | 格子面积的40% | 格子面积的40% |
| 图标位置 | 格子中心 | 格子中心 | 格子中心 |
