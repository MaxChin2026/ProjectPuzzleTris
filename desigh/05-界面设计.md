# 05 — 界面设计

## 竖屏整体布局

```
┌──────────────────────────────────────┐ ← 屏幕顶部
│[🧙Lv] ⏸ [分数/波次]  ❤️❤️❤️  [阶段]  │ ← 顶部HUD栏（约5%，左上英雄头像+经验环）
├──────────────────────────────────────┤
│  ████████████████░░░░   增行倒计时    │ ← 底部增行进度条（紧贴HUD下方，约3%）
├────┬──────────────────────────┬──────┤
│    │  → → → → → → → → → →    │      │ ← 顶边路径（敌人从左往右穿越）
│ ↑  ├──────────────────────────┤  ↓  │
│ 👾 │                          │  👾 │ ← 主棋盘（约62%）
│ 👾 │      棋盘区域             │  👾 │   左边 = 敌人从下往上行进（↑）
│    │     （10 × 20 格）        │     │   右边 = 敌人从上往下行进（↓）
│ 👾 │                          │  👾 │
│ ↑  ├──────────────────────────┤  ↓  │
│[起]│                           │[终] │ ← 起点（左下角）→ 终点（右下角）
├────┴──────────────────────────┴──────┤
│  ┌───────┐ ┌───────┐ ┌───────┐ ┌───────┐ ┌───────┐   │ ← 方块槽区域（约25%）
│  │ 方块A│    │ 方块B│    │ 方块C│    │ 方块D│    │ 方块E│   │   5选1候选方块
│  │[🎯图] │    │      │    │      │   │   （示意：技能格图标叠加在格子上）
│  └──────┘    └──────┘    └──────┘   │
└──────────────────────────────────────┘ ← 屏幕底部
```

> [!note] 敌人路径方向说明
> 敌人从棋盘**左下角外侧**（**起点**）出发，路径如下：
> 1. **第一段**：沿棋盘**左边**，从下方**往上**行进（起点→左上角）
> 2. **第二段**：沿棋盘**顶边**，从左**往右**行进（左上角→右上角）
> 3. **第三段**：沿棋盘**右边**，从上方**往下**行进（右上角→终点）
> 4. **终点**：棋盘**右下角外侧**，到达后扣玩家生命值
>
> 敌人路径绘制于棋盘边框**外侧覆盖层**，不占用棋盘内部格子空间。

## 各区域设计规范

### 顶部 HUD 栏

| 元素 | 位置 | 内容 |
| ---- | ---- | ---- |
| 英雄头像 | 最左侧 | 圆形头像，外围环形经验条（顺时针金色，满时白色脉冲） |
| 英雄等级 | 头像右下角 | 小圆角数字标「Lv.X」 |
| 暂停按钮 | 头像右侧 | 暂停/菜单 |
| 当前分数 | 中左 | 数字大字显示，消行时弹出加分动画 |
| 生命值 | 中部 | ❤️图标排列，每个图标代表1点生命 |
| 波次信息 | 中右 | "第 3 波 / 5 波" |
| 难度阶段 | 右侧 | "阶段 2 / 5" |

### 棋盘区域

| 元素 | 设计 |
| ---- | ---- |
| 棋盘网格线 | 细线，低对比度，不干扰方块可读性 |
| 活动方块 | 实色填充，每种形状对应固定颜色 |
| Ghost Piece | 同色半透明（约30%不透明度），显示落点预判 |
| 技能格图标 | 在含技能格的格子上叠加技能图标（🎯/💥/❄️等），图标占格子面积1/3 |
| 满棋盘警告 | 方块接近顶部时，棋盘顶部边框红色脉冲闪烁 |
| 消行动画 | 整行高亮后碎裂飞出，方块格变为彩色子弹粒子 |
| 子弹弧线 | 消行产生的子弹沿贝塞尔曲线飞向棋盘边沿的敌人，约 300ms |

### 敌人路径层（叠加于棋盘边框外侧）

| 元素 | 设计 |
| ---- | ---- |
| 路径轨道 | 棋盘左边/顶边/右边外侧约 1~2 格宽度的路径区域 |
| 敌人图形 | 简洁像素风小怪图标，HP条显示在图标上方 |
| 敌人移动 | 匀速沿路径行进，受技能效果时有视觉减速/停止动效 |
| 死亡动效 | 被子弹击中 HP 归零时，敌人爆炸消失粒子特效 |
| 受击反馈 | 被子弹命中时，敌人图标短暂闪白 + HP 条更新 |
| 接近终点警告 | 敌人到达右边靠近终点约20%位置时，路径区域边框橙红色脉冲 |
| 漏怪动效 | 敌人到达终点时，生命图标（❤️）闪烁消失，震屏即时反馈 |

### 底部增行进度条

| 元素 | 设计 |
| ---- | ---- |
| 进度条 | 横向满格逐渐耗尽的倒计时条 |
| 颜色变化 | 绿色（充足）→ 黄色（警告）→ 红色（即将增行） |
| 满格触发 | 进度归零时插入增行，进度条重置 |

### 方块槽区域

| 元素 | 设计 |
| ---- | ---- |
| 槽位背景 | 圆角矩形边框，区分选中/未选中状态 |
| 候选方块 | 实色渲染，与棋盘中使用的颜色一致 |
| 技能格标识 | 在含技能格的格子叠加对应技能图标 |
| 选中状态 | 槽位高亮边框 + 轻微放大动画 |
| 新方块入场动画 | 轻微弹入效果（scale 0.8→1.0，约150ms） |

### 英雄升级面板（覆盖全局）

> [!info] 触发时机
> 英雄经验环填满时，游戏**立即暂停**，弹出英雄技能选择面板。

```
┌──────────────────────────────────────────┐
│  ✨ 英雄升级！选择一种技能              │  ← 标题居中，含升至Lv.X提示
│                                          │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐│
│  │  🔥      │  │  ❄️      │  │  🔗      ││
│  │ 烈焰弹道 │  │ 寒冰领域 │  │ 连消加成 ││
│  │          │  │          │  │          ││
│  │每格伤害+1│  │路径敌人  │  │3秒内连消 ││
│  │          │  │持续-20%  │  │+5追踪弹  ││
│  │ 🔴输出流 │  │ 🔵控制流 │  │ 🟢消除流 ││
│  └──────────┘  └──────────┘  └──────────┘│
│              [已持有技能 2/5]             │
└──────────────────────────────────────────┘
```

| 元素 | 设计 |
| ---- | ---- |
| 遮罩 | 全屏半透明黑色遮罩（opacity: 0.7） |
| 卡片布局 | 3张卡片横排，各代表一种构筑套路的技能 |
| 构筑标签 | 卡片底部颜色徽章（🔴/🔵/🟢）标注构筑归属 |
| 选中动画 | 点击后被选中卡片放大+高亮，其余渐出 |
| 恢复游戏 | 选择后0.5秒过渡动画，游戏自动继续 |
| 持有进度 | 面板底部显示「已持有技能 X/5」 |

## 视觉风格方向

| 要素 | 方向 |
| ---- | ---- |
| 整体风格 | 轻科技感扁平风 / 清晰可读优先 |
| 配色 | 深色背景，方块高饱和度亮色，敌人路径区域暗色对比 |
| 字体 | 无衬线，中英文混排清晰 |
| 动效 | 轻量流畅：子弹弧线飞行 + 碎裂特效 + 敌人死亡爆炸 |
| 技能释放特效 | 每个技能有专属视觉效果（冰冻结晶/爆炸球/闪电链等）|
